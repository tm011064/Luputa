Description: The CommonTools.Components.Flash.OpenFlashChart namespace acts as an API for teethgrinder's Open-Flash-Chart flash project
(Version 2 -> http://teethgrinder.co.uk/open-flash-chart-2/). The CommonTools.Components.Flash.OpenFlashChart.Chart class exposes a 
ToString() method which writes a specified JSON object based on its properties. This JSON object can be interpreted by the open-flash-chart.swf movie 
to draw the chart.
